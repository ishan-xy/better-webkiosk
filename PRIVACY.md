# Privacy Policy — Better WebKiosk

**Last updated:** *December 28, 2025*

Better WebKiosk is a Chrome extension that enhances Thapar University’s WebKiosk by providing secure auto-login and a local marks summary.


## Data Collection

Better WebKiosk **does not collect, store, or share any personal data** with the developer or any third party.


## Local Storage

The extension may store the following information **locally on the user’s device only**:

* Enrollment Number
* WebKiosk Password / PIN (stored in encrypted form)

This data is used solely to enable the auto-login feature and never leaves the user’s browser.


## Academic Data

Marks and academic information are read directly from the WebKiosk webpage and processed **entirely within the browser**.
No academic data is stored, logged, or transmitted.


## Network Usage

Better WebKiosk does **not** make any external network requests and does not communicate with any servers.


## Permissions

The extension uses Chrome permissions only to:

* Store encrypted credentials locally
* Interact with the WebKiosk website

Permissions are not used for tracking, analytics, or advertising.


## Data Removal

Users can clear all stored data at any time using the extension’s interface.
Uninstalling the extension removes all data automatically.


## Changes

Any changes to this policy will be published on the project’s GitHub repository.

Better WebKiosk complies with the Chrome Web Store User Data Policy and does not collect or transmit personal or sensitive user data.
